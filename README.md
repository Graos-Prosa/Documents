# 📑 Grãos & Prosa - Documentação

Este repositório reúne toda a documentação do projeto **Grãos & Prosa**, incluindo artefatos de análise, modelagem, arquitetura e apoio ao desenvolvimento.

O objetivo é centralizar informações para **consulta, acompanhamento e validação** das decisões tomadas durante o projeto.
